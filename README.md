# Chatbot de Soporte Técnico

Proyecto realizado para la materia Organización Empresarial.

## Descripción

El sistema simula un chatbot de soporte técnico basado en BPMN 2.0 utilizando Python y SQLite.

El chatbot permite:

* Validar empleados
* Registrar tickets
* Gestionar niveles de urgencia
* Consultar soluciones automáticas
* Derivar casos críticos

## Tecnologías utilizadas

* Python
* SQLite
* BPMN 2.0
* GitHub

## Cómo ejecutar

1. Ejecutar:

```bash
python init_db.py
```

2. Luego ejecutar:

```bash
python main.py
```
